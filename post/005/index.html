AMD 黑苹果关于本机中处理器 CPU 显示错误更改

2020-01-16
黑苹果

文章目录

  • 如果您的cpu只显示频率（四叶草clover情况）
  • 如果您的cpu已经有信息，但是显示为intel的。
  • 备注
  • 其他

如果您的cpu只显示频率（四叶草clover情况）

  • 使用CloverConfigur（有中文版）挂载分区，打开EFI/CLOVER目录下的config.plist

  • CPU设置里面填写CPU类型，例如填写0X1001，保存（win+s），重启。

  • 查看显示信息，应该是显示intel的。

  • 打开终端 输入命令： sudo -s 回车输入用户密码密码，再回车， 输入命令： sudo mount -u -o rw / 回车

  • 右键访达-前往文件夹，输入 /System/Library/PrivateFrameworks/AppleSystemInfo.framework/Versions/A/Resources/zh_CN.lproj/

  • 找到processors.strings文件，右键显示简介，点击右下角解锁，然后把只读改成读写

  • App Store下载BBEdit，然后用来打开 processors.strings文件，复制本机信息，例如（6核Intel Core i5），win+f，查找到所在位置，改成想要显示的型号保存

  • 保存，重启，完事。

如果您的cpu已经有信息，但是显示为intel的。

  • 打开终端 输入命令： sudo -s 回车输入用户密码密码，再回车， 输入命令： sudo mount -u -o rw / 回车

  • 右键访达-前往文件夹，输入 /System/Library/PrivateFrameworks/AppleSystemInfo.framework/Versions/A/Resources/zh_CN.lproj/

  • 找到processors.strings文件，右键显示简介，点击右下角解锁，然后把只读改成读写

  • App Store下载BBEdit，然后用来打开 processors.strings文件，复制本机信息，例如（6核Intel Core i5），win+f，查找到所在位置，改成想要显示的型号保存

  • 保存，重启，完事。

备注

前面的步骤已经做完，但是更改processors.strings只读成读写的时候提示没有权限

  • 打开终端 输入命令： killall Finder

  • 找到processors.strings文件，把文件拷贝到桌面，App Store下载BBEdit，然后用来打开 processors.strings文件，复制本机信息，例如（6核Intel Core i5），win+f，查找到所在位置，改成想要显示的型号保存

  • 桌面的processors.strings文件拖拽回原来的位置，点击【认证】，再点击【替换】，再输入开机密码

重启，完事。
